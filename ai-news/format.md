# 最终输出格式

`ai-news` 的最终结果必须输出为 markdown，并严格遵守下面的结构。不要输出线性列表，不要省略发布时间和来源链接。

## 标题

```markdown
# AI 新闻简报｜2026-03-29
```

如果用户指定了日期范围，标题要写绝对日期，例如：

```markdown
# AI 新闻简报｜2026-03-28
```

或：

```markdown
# AI 新闻简报｜2026-03-22 至 2026-03-29
```

## 正文模板

```markdown
# AI 新闻简报｜{date}

## 大模型

### {文章标题}
- 摘要：{中文摘要，1 到 3 句话，保留关键信息}
- 发布时间：{优先写成 x 小时前 / x 天前；如有必要可补充绝对时间，例如 3 小时前（2026-03-29 14:30）}
- 来源链接：{URL}
```

## 成稿规则

1. 一级标题只能是整份简报标题。
2. 分类用二级标题。
3. 文章标题用三级标题。
4. 每条新闻固定三行字段：
   - `摘要`
   - `发布时间`
   - `来源链接`
5. `来源链接` 优先使用原始报道链接，而不是二次聚合页。
6. 日报场景下，发布时间优先写相对时间，例如 `2 小时前`；如果有绝对时间，建议补在括号里。
7. 同一分类下按重要性排序，不要求严格按时间倒序。
8. 如果某个分类当天没有足够有效新闻，可以整体省略该分类。
9. 结尾不要再附“过程说明”“抓取日志”“搜索命令回显”。

## 示例

```markdown
# AI 新闻简报｜2026-03-29

## 大模型

- 摘要：OpenAI 发布了新一代推理模型，重点提升复杂任务下的推理稳定性与工具调用能力，并同步开放给开发者使用。

- 来源链接：https://example.com/openai-model

## 开源

### 新的开源 Agent 框架发布
- 摘要：一款新的开源 Agent 框架发布，支持多工具编排、记忆管理和可观测性，目标是降低复杂工作流搭建门槛。

- 来源链接：https://example.com/open-source-agent
```
