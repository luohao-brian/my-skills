# 最终输出格式

`ai-news` 的最终结果必须输出为 markdown，并严格遵守下面的结构。

## 标题

```markdown
# AI 新闻简报｜2026-03-29
```

如果用户指定了日期范围，标题要写绝对日期，例如：

```markdown
# AI 新闻简报｜2026-03-28
```

或：

```markdown
# AI 新闻简报｜2026-03-22 至 2026-03-29
```

## 排版规则

1. 一级标题只能是整份简报标题。
2. 分类用二级标题。
3. 文章标题用三级标题。
4. 每条新闻固定三行字段：
   - `摘要：`
   - `发布时间：`
   - `来源链接：`
5. 标题、分类标题、条目标题、三行字段都必须独占一行。
6. 每个分类标题前后保留空行。
7. 每个条目标题前后保留空行。
8. 每个条目的三行字段连续排列，不使用列表符号，不嵌套列表，不插入额外说明行。
9. 不使用表格、引用块、任务列表、HTML、内联标题、把多个条目写进同一段落。
10. 结尾不要附“过程说明”“抓取日志”“搜索命令回显”。
