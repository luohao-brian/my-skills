# AI 新闻简报

产出中文版 AI 新闻简报。重点不是“搜到一些链接”，而是按稳定 workflow 拿到尽量一手、有效、准确的信息，再输出统一格式。

先读：

- [sources.md](sources.md)：官方优先源、媒体补充源、搜索补漏策略
- [format.md](format.md)：最终输出格式

## 目标

生成一份 markdown 简报，满足：

1. 必须按分类组织，不允许只输出线性列表。
2. 每条新闻必须包含标题、摘要、发布时间、来源链接。
3. 标题和摘要用中文。
4. 同一条新闻只能出现一次，不要跨分类重复。
5. 优先引用官网、官方博客、官方产品页、官方社媒，而不是媒体转述。

## 默认时间窗口

如果用户没有指定日期，默认整理最近 `24-48 小时` 内仍有参考价值的 AI 新闻。

时间判断规则：

- 默认先看最近 48 小时
- 如果 48 小时内信息太少，再明确说明覆盖不足，不要偷偷放宽到更久
- 如果用户指定“今日 / 昨日 / 本周”，按用户要求执行，并在标题里写绝对日期
- 发布时间展示优先用相对时间，例如 `3 小时前`；如果有绝对时间，建议写成 `3 小时前（2026-03-29 14:30）`

## 分类

- 大模型
- Agent
- 融资 / 商业
- 安全 / 治理
- 应用 / 产品
- 开源

如果某个分类在目标时间窗内没有足够有效内容，可以省略；但必须先完成官方源扫描和搜索补漏，不能先入为主地判定“没有新闻”。

## 工作流

### 1. 先扫官方源

先读 [sources.md](sources.md) 里的“官方一级优先源”。

优先级固定为：

1. AI 公司官网、官方博客、官方产品页、官方公告页
2. 官方 X / Twitter 等官方社媒渠道
3. 媒体、聚合站、RSS
4. 搜索补漏

对 OpenAI、xAI 这类存在 Cloudflare 或反爬挑战的站点：

- 不要因为简单 HTTP 失败就判定源站失效
- 先尝试 `url-to-markdown` / `web-fetch`
- 仍不稳定时，再用 `volc-websearch` 对官方域名和官方账号做定向搜索

### 2. 再扫媒体与聚合源

再读 [sources.md](sources.md) 中的媒体与聚合补充源。

这些源用于：

- 补官方源没及时发出的产品更新
- 发现中文侧热点和转载链路
- 补融资、合作、生态消息

媒体报道默认是补充材料。只要官方源里有一手公告，成稿时优先引用官方链接。

### 3. 用搜索做补漏和交叉验证

固定源之后，必须再做搜索补漏。不要停留在“官网 + 媒体”两层。

最低要求：

- 至少覆盖 `tavily`、`bocha` 两个 provider
- 对英文国际动态或需要英文交叉验证的分类，再补 `brave`
- `volc` 是可选搜索引擎，用于中文站点、日期窗口和站点白名单补充验证；可用时优先补上
- 每个分类至少做一轮中文 query 和一轮英文 query
- 对优先公司名单，再显式做一轮“官方域名 + 产品名”定向检索

搜索策略：

1. 先做官方站点定向检索
2. 再做宽召回
3. 最后做站点 / 日期 / 语言定向检索

不要只搜 `AI news` 这种宽泛 query。要拆成：

- 公司名 + 产品名
- 分类主题词
- 日期词面 query

当 provider 不支持时间过滤时，补一轮词面日期 query，例如：

- `2026-03-29 AI agent news`
- `2026年3月29日 大模型 发布`

### 4. 抓详情页，补齐字段

候选新闻进入成稿前，要尽量补齐：

- 中文标题
- 中文摘要
- 发布时间
- 最终来源链接

列表页字段不完整时，必须抓详情页确认；不要拿列表页上的模糊文案直接成稿。

### 5. 过滤、去重、校验

去重优先顺序：

1. 最终来源链接
2. 归一化英文原标题
3. 中文标题里的核心实体 + 动作组合

筛选原则：

- 只保留 AI 相关内容
- 同源重复报道只保留信息最完整的一条
- 二手转载优先让位于原始来源或一手报道
- 无发布时间、无正文支撑、明显营销软文不要收录

准确性要求：

- 优先使用官网、官方博客、官方产品页和官方社媒
- 对发布时间不清楚的条目，不要硬写日报日期冒充发布时间
- 对来源不清楚的聚合页，尽量追到原始来源链接
- 对有争议或只有单点来源的信息，至少再做一轮搜索交叉验证

## 输出要求

最终输出必须严格遵守 [format.md](format.md)。

最低标准：

- 分类是一级结构
- 每条包含标题、摘要、发布时间、来源链接
- 发布时间在日报场景下优先用相对时间表达
- 不输出抓取日志、命令回显、过程噪音
