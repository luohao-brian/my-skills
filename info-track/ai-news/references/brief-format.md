# AI 新闻格式

最终结果输出为 markdown。

## 标题

```markdown
# AI 新闻简报｜2026-03-29
```

如果用户指定日期范围，标题写绝对日期。

## 正文模板

```markdown
# AI 新闻简报｜{date}

## 🧠 模型 / 研究

### {中文标题}
摘要：{中文摘要}
来源链接：{URL}

## 🛠️ Agent / 开发者工具

### {中文标题}
摘要：{中文摘要}
来源链接：{URL}

## 🚀 产品 / 应用

### {中文标题}
摘要：{中文摘要}
来源链接：{URL}

## 💼 商业 / 融资

### {中文标题}
摘要：{中文摘要}
来源链接：{URL}

## 🛡️ 安全 / 治理

### {中文标题}
摘要：{中文摘要}
来源链接：{URL}
```

## 排版

1. 一级标题只能是整份简报标题。
2. 分类用二级标题，标题固定带 emoji。
3. 文章标题用三级标题。
4. 每条新闻固定两行字段：`摘要：`、`来源链接：`。
5. 标题、分类标题、条目标题和字段都独占一行。
6. 不使用表格、引用块、任务列表、HTML、内联标题，结尾不附过程说明。
